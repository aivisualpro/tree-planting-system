# Admin User Guide - Tree Planting Web Portal

Welcome to the Admin Portal. This guide outlines how to manage users, monitor sync health, and extract data via the web dashboard.

## 1. Accessing the Portal
1. Navigate to `https://admin.treeplanting.com`.
2. Log in using your Manager or Admin credentials.
3. You will be redirected to the **Dashboard**, which provides a high-level overview of total plantings and active users.

## 2. User Management
1. Navigate to **Users** in the sidebar.
2. **Add User**: Click "New User", enter their details, and assign a role (e.g., `field_user`, `manager`).
3. **Assign Regions**: Users must be assigned to specific countries/regions to download the correct data.
4. **Deactivate**: Click the "..." next to a user to disable their access immediately.

## 3. Monitoring Visits & Sync Health
1. Navigate to **Visits**. Here you can see a real-time table of all synced data.
2. **Review Photos**: Click on a visit row to open the details pane and verify planting photos.
3. **Data Quality**: Filter by "Missing Photos" or "Flagged Locations" to audit field performance.

## 4. Managing Translations
The app supports dynamic translations (English, French, Swahili).
1. Navigate to **Settings > Translations**.
2. Search for a term (e.g., "Save button").
3. Edit the translation for the target language.
4. Changes will automatically sync to field workers' devices upon their next connection.

## 5. Reviewing Field Feedback
Field workers can submit bug reports directly from the mobile app.
1. Navigate to **Support > Feedback**.
2. Review the user's comment, device info (OS, app version), and attached screenshots.
3. Update the ticket status (New -> Investigating -> Resolved) to track progress.

## 6. Exporting Data
1. Navigate to **Reports**.
2. Select your date range and region.
3. Click **Export CSV** to download the dataset for external analysis.
